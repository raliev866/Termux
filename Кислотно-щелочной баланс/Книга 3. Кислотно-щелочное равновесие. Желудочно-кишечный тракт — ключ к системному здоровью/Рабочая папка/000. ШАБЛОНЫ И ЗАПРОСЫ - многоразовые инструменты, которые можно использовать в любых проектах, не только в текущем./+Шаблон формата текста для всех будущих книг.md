# ШАБЛОН ФОРМАТА ТЕКСТА
## Единый стандарт оформления книг серии «Кислотно-щелочное равновесие»

---

## 1. ОБЩАЯ СТРУКТУРА ТЕКСТА

### 1.1. Иерархия заголовков

```
# Название книги (Уровень 1) — только в начале файла
# Название раздела (Уровень 1)
## Глава [номер]. Название главы (Уровень 2)
### [номер подглавы]. Название подглавы (Уровень 3)
#### Второстепенный блок (Уровень 4 — использовать редко)
```

**Правила:**
- **Уровень 1 (#)** — название книги (только в начале файла) и названия разделов
- **Уровень 2 (##)** — названия глав (Глава 1, Глава 2...)
- **Уровень 3 (###)** — подглавы (1.1, 1.2, 1.3...)
- **Уровень 4 (####)** — только для сложных глав, где нужно дробить глубже (1.1.1, 1.1.2...)

**Пример:**
```
# Кислотно-щелочное равновесие: Желудочно-кишечный тракт — ключ к системному здоровью

# Раздел I. Желудок: кислотный барьер организма

## Глава 1. Физиология желудочного сока

### 1.1. Состав желудочного сока
### 1.2. Париетальные клетки
### 1.3. Протонная помпа
```

## 2. ОФОРМЛЕНИЕ ТЕКСТА

### 2.1. Выделения

| Элемент | Оформление | Пример |
|---------|------------|--------|
| **Ключевые термины** | `**жирный**` | Соляная кислота (**HCl**) вырабатывается... |
| *Акцент, иностранные слова* | `*курсив*` | Этот процесс называется *апоптозом*... |
| ***Особо важное*** | `***жирный курсив***` | ***Запомните: без HCl невозможно усвоение B12*** |
| `Код / команды` | \`обратные кавычки\` | Введите команду `генератор` |

**Правила:**
- **Жирный шрифт** — только для ключевых терминов при первом упоминании
- *Курсив* — для иностранных слов, названий, акцентов
- ***Жирный курсив*** — только для критически важных выводов (1-2 раза на главу)

### 2.2. Списки

**Маркированный список (основной):**
```
- Первый пункт
- Второй пункт
  - Подпункт (с отступом 2 пробела)
  - Ещё подпункт
- Третий пункт
```

**Нумерованный список (последовательность):**
```
1. Первый шаг
2. Второй шаг
   1. Подшаг 2.1
   2. Подшаг 2.2
3. Третий шаг
```

**Правила:**
- После маркера списка (`-` или `1.`) — **один пробел**
- Между пунктами списка — **пустая строка не нужна** (кроме сложных случаев)
- Если пункт длинный (больше 3 строк) — разделяй на подпункты

### 2.3. Цитаты и примеры

```
> Это цитата из исследования или важное утверждение.
> Она выделяется таким блоком.

> **Важно:** Если внутри цитаты нужно выделение — используй **жирный**.
```

### 2.4. Чек-листы (специальный формат)

В конце каждой главы — **обязательный чек-лист**:

```
## ✅ Чек-лист: что запомнить из главы

- [ ] Пункт первый для самопроверки
- [ ] Пункт второй для самопроверки  
- [ ] Пункт третий для самопроверки

> **На заметку:** Если есть важное дополнение к чек-листу
```

**Почему так:**
- Чек-лист — единственное завершение главы (никаких «выводов» или «резюме»)
- Читатель видит, что усвоил, а что нужно повторить

## 3. ОФОРМЛЕНИЕ ТАБЛИЦ (ОСОБЫЙ СЛУЧАЙ)

### 3.1. Общее правило

**Таблицы — только в электронной версии.** Для аудиоформата у каждой таблицы должен быть **текстовый дубль**.

### 3.2. Формат таблицы в Markdown

```
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|-------------|-------------|-------------|
| Ячейка 1.1  | Ячейка 1.2  | Ячейка 1.3  |
| Ячейка 2.1  | Ячейка 2.2  | Ячейка 2.3  |
```

### 3.3. Текстовый дубль таблицы (для аудио)

Сразу после таблицы идёт **текстовый дубль** в таком формате:

```
📋 **Текстовая версия таблицы для прослушивания:**

Таблица [Название] содержит следующие данные:
- По строке 1: [пояснение]
- По строке 2: [пояснение]
- Ключевой вывод из таблицы: [вывод]
```

**Пример:**
```
📋 **Текстовая версия таблицы для прослушивания:**

Таблица «Время гибели патогенов при pH 2,0» показывает:
- Холерный вибрион погибает за 5–10 минут
- Сальмонелла — за 10–20 минут
- Стафилококк — за 20–40 минут

Главный вывод: даже кратковременное снижение кислотности делает организм уязвимым для кишечных инфекций.
```

## 4. РАБОТА С ТЕРМИНАМИ

### 4.1. Первое упоминание термина

При первом упоминании сложного термина:
1. Выдели его **жирным**
2. Дай краткое объяснение в скобках или через тире

**Пример:**
> Главную роль играют **париетальные клетки** (обкладочные клетки желудка, вырабатывающие соляную кислоту).

### 4.2. Иностранные термины

При первом упоминании давай оригинал в скобках:
> Синдром избыточного бактериального роста (**СИБР**, от англ. *SIBR — Small Intestinal Bacterial Overgrowth*)

### 4.3. Аббревиатуры

При первом упоминании — полностью, потом можно сокращать:
> Соляная кислота (**HCl**) играет ключевую роль. Далее мы будем говорить просто HCl.

## 7. ТРЕБОВАНИЯ К АБЗАЦАМ

### 7.1. Длина абзаца

- **Идеально:** 3–5 предложений
- **Максимум:** 7–8 предложений (если тема сложная и требует развития)
- **Минимум:** 1 предложение (для акцента или связки)

### 7.2. Структура абзаца

1. **Первое предложение** — тема абзаца (что сейчас будем обсуждать)
2. **Середина** — развитие мысли, аргументы, примеры
3. **Последнее предложение** — вывод или переход к следующей мысли

**Пример:**
> **Ключевую роль в производстве HCl играют париетальные клетки.** Эти клетки находятся в слизистой оболочке тела желудка и дна желудка. В них происходит удивительный процесс: из углекислого газа и воды образуется угольная кислота, которая распадается на ион водорода и бикарбонат. Ион водорода с помощью протонной помпы перекачивается в просвет желудка в обмен на ион калия. **Так создаётся та самая кислая среда, необходимая для пищеварения.**

### 7.3. Связки между абзацами

Используй слова-связки:
- «Однако...»
- «С другой стороны...»
- «Важно понимать...»
- «Рассмотрим подробнее...»
- «Отсюда следует вывод...»

## 8. РИТМИЧЕСКИЙ РИСУНОК

### 8.1. Чередование длины предложений

Для лёгкости восприятия чередуй:
- **Короткие предложения** (3–7 слов) — для акцентов
- **Средние предложения** (8–15 слов) — основной поток
- **Длинные предложения** (16–25 слов) — для сложных мыслей (редко)

### 8.2. Визуальные паузы

- Пустые строки между абзацами — **обязательны**
- Пустые строки перед важными выводами — **желательны**
- Пустые строки перед чек-листом — **обязательны**

## 10. ПРИМЕР ИДЕАЛЬНОЙ ГЛАВЫ (ЭТАЛОН)

```
# Раздел I. Желудок: кислотный барьер организма

## Глава 4. Гипохлоргидрия (пониженная кислотность желудка)

### 4.1. Что это такое?

**Гипохлоргидрия** — это состояние, при котором желудок вырабатывает недостаточно соляной кислоты (HCl). В лёгких случаях говорят о пониженной кислотности, в тяжёлых — об ахлоргидрии (полном отсутствии HCl).

💡 **Знаете ли вы?** После 40 лет уровень кислотности снижается у каждого второго человека, а после 60 — у 70–80%.

### 4.2. Как распознать?

Симптомы гипохлоргидрии часто путают с повышенной кислотностью. Вот основные признаки:

- Тяжесть после еды, особенно мясной
- Отрыжка с тухлым запахом
- Вздутие через 1–2 часа после еды
- Чувство, что еда «стоит камнем»
- Непереносимость мяса (хочется есть, но после приёма тяжело)
- Хронические запоры или диарея

> ⚠️ **Важно:** Изжога тоже может быть симптомом низкой кислотности! Когда привратник плохо закрывается, содержимое забрасывается в пищевод и вызывает жжение.

### 4.3. Почему это опасно?

Недостаток HCl ведёт к цепочке проблем:

1. Белки не расщепляются полностью
2. Не активируется пепсин
3. Не поступает сигнал поджелудочной железе
4. Желчь застаивается
5. В тонкий кишечник попадают непереваренные частицы и патогены

**Результат:** СИБР, дефициты витаминов (особенно B12 и железа), хроническое воспаление.

## ✅ Чек-лист: что запомнить из главы 4

- [ ] Гипохлоргидрия — это пониженная кислотность желудка
- [ ] После 40 лет риск гипохлоргидрии резко возрастает
- [ ] Симптомы: тяжесть после еды, отрыжка, вздутие
- [ ] Изжога может быть признаком низкой кислотности
- [ ] Недостаток HCl ведёт к СИБР и дефициту витаминов
```
