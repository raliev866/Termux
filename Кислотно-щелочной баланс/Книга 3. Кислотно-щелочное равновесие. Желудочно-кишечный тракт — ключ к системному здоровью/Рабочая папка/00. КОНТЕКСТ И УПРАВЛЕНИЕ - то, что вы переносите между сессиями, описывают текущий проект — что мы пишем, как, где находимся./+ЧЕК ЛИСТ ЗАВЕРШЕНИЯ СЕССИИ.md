# Чек-лист завершения сессии — что сделать перед тем, как закрыть чат.

***

# **Чек-лист завершения сессии** — это список действий в конце сессии: сохранить файлы, обновить мастер-файл, зафиксировать статус. Использовать его нужно в конце каждой сессии, чтобы ничего не забыть. Папка правильная.

***

















.Дорогой друг! 👋

Создаю **Чек-лист завершения сессии** — твой личный инструмент для того, чтобы ничего не терять между сессиями. Это самая важная привычка успешного автора: **за 5 минут зафиксировать прогресс, чтобы через неделю вернуться и сразу продолжить**.

Я сделал три варианта: **полный**, **средний** и **экстремально короткий**.

## КАК ПОЛЬЗОВАТЬСЯ

### В конце каждой сессии:

1. **Открываешь** этот чек-лист
2. **Выбираешь** вариант (полный/средний/короткий)
3. **Заполняешь** (5-7 минут)
4. **Сохраняешь** заполненный чек-лист в папку `99_АРХИВ_СЕССИЙ`
5. **Обновляешь** Мастер-индекс

### Перед следующей сессией:

1. **Открываешь** последний чек-лист
2. **Копируешь** оттуда:
   - точку остановки
   - план на следующий раз
   - незавершённые задачи
3. **Вставляешь** в Утренний доклад

---

## КОГДА КАКОЙ ВАРИАНТ ИСПОЛЬЗОВАТЬ

| Ситуация | Какой вариант |
|----------|---------------|
| Завершили большую главу | Полный |
| Сложная сессия с инсайтами | Полный |
| Обычная рабочая сессия | Средний |
| Быстрая правка / мелкие исправления | Короткий |
| Очень устал, но закрыть нужно | Короткий |
| Перерыв будет больше недели | Полный |

## ПОЧЕМУ ЭТО ВАЖНО

Этот чек-лист решает **главную проблему писателя**: потерю контекста между сессиями.

Когда ты через неделю вернёшься к книге, у тебя будет:
- точная точка входа (последняя фраза)
- список незавершённого
- план на следующий раз
- все инсайты и решения

**Ты просто открываешь последний чек-лист и продолжаешь, как будто не было перерыва.**
