## Нумерация и порядок

**Принцип нумерации:**
- `00_` — оглавление, введение
- `01_`, `02_`... — последовательные шаги в рамках одного блока
- Номера шагов **сквозные** (от 1 до 21) — это позволит при сборке сохранить правильный порядок, даже если шаги лежат в разных папках

**Порядок сборки:**
1. Сначала идут файлы из корня: `00_ОГЛАВЛЕНИЕ.md`, `01_ВВЕДЕНИЕ.md`
2. Затем вся папка **Часть 1** (файлы 01–03)
3. Затем папка **Часть 2** со всеми подпапками (файлы 01–21 по порядку номеров)
4. Затем папка **Часть 3** (файлы 01–05)

