## Пояснения для Word:

Если вы настраиваете **шаблон reference.docx** в Word, вот как найти эти стили:

- **Заголовок 1-9** — стили для заголовков разных уровней
- **Основной текст** — обычный текст документа
- **Цитата** — для выделенных цитат
- **Сноска** — для примечаний внизу страницы
- **Название таблицы** — подпись над или под таблицей
- **Название рисунка** — подпись к изображениям
- **Термин** и **Определение** — для списков определений

Это поможет вам настроить идеальный шаблон для конвертации!
