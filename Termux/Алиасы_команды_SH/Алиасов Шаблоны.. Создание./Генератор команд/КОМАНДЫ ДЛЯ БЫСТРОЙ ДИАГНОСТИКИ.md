

# КОМАНДЫ ДЛЯ БЫСТРОЙ ДИАГНОСТИКИ
___

### Проверить все команд
```bash
# Проверить все команды
ls -la ~/bin/
echo $PATH | grep "$HOME/bin"
cat ~/.bashrc | grep alias
```
___

### Посмотреть логи

```bash
# Посмотреть логи
tail -20 ~/.termux_scripts/commands.log
```
___

```bash
генератор
```
___

### Если что-то не работает

```bash
# Если что-то не работает
source ~/.bashrc

```
