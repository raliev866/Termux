## 2. СТИЛЬ И ФОРМАТ (жёсткие правила)

### Основные принципы

| Правило | Описание |
|---------|----------|
| **Научно-популярный стиль** | Сложные вещи объясняем простым языком, без занудства и излишней академичности |
| **Метафоры и образы** | Используем для объяснения механизмов (печень — завод, желчь — мусоровоз, pH — костёр) |
| **Практическая направленность** | После теории обязательно "что делать" |
| **Безопасность** | Всегда указываем риски и противопоказания |
| **Таблицы** | ❌ **НЕ ИСПОЛЬЗУЕМ**. Заменяем текстом, списками, выделениями |
| **Сквозная тема** | Каждая глава связана с кислотно-щелочным балансом (явно или неявно) |

## 5. ЧАСТЫЕ ОШИБКИ (чего избегать)

| Ошибка | Как правильно |
|--------|---------------|
| Слишком сложно, много терминов | Объяснять каждый термин при первом упоминании |
| Нет связи с pH | Даже в главе про холод напомнить, как это влияет на кислотность |
| Только теория, нет практики | Всегда давать "инструкцию к применению" |
| Забыть про риски | Любой метод может навредить — предупредить |
| Использовать таблицы | Заменять текстовыми списками |
| Противоречить предыдущим главам | Проверять согласованность |
