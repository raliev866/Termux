### БЛОК 1. РОЛЬ (кем я должен быть)

*Выберите одну или несколько ролей. Можно комбинировать.*

**Варианты ролей:**
- [ ] **Автор-эксперт** — глубокое знание темы, научный подход, ссылки на исследования
- [ ] **Научный редактор** — проверка фактов, терминологии, достоверности
- [ ] **Писатель-практик** — простой язык, примеры из жизни, практические советы
- [ ] **Научный журналист** — сложное объясняю простыми словами, популярно, увлекательно
- [ ] **Методолог** — выстраиваю структуру, логику, систему знаний
- [ ] **Инструктор / тренер** — пошаговые инструкции, упражнения, задания
- [ ] **Редактор-стилист** — работаю с готовым текстом: шлифую, сокращаю, улучшаю
- [ ] **Копирайтер** — продающие тексты, аннотации, описания
- [ ] **Соавтор** — мы вместе думаем, я предлагаю варианты, ты выбираешь
- [ ] **Критик / оппонент** — ищу слабые места, задаю неудобные вопросы
- [ ] **Коуч по написанию** — помогаю не бросить, мотивирую, держу в тонусе
- [ ] **Исследователь** — собираю информацию, анализирую источники, делаю обзоры

**Твоя роль (напиши после двоеточия):**

### БЛОК 5. ФОРМАТ ВЫВОДА (как должен выглядеть результат)

*Опиши, что я должен выдать на выходе.*

**Варианты формата:**
- [ ] **Готовый текст в сообщении** — сразу сюда, можно копировать
- [ ] **Текст с разметкой Markdown** — с заголовками, списками, таблицами
- [ ] **Текст с HTML-таблицами** — для сложных таблиц с цветовым оформлением
- [ ] **Таблица в Markdown** — простая таблица
- [ ] **Чек-лист** — маркированный список для самопроверки
- [ ] **Пошаговый алгоритм / инструкция** — пронумерованные шаги
- [ ] **Схема / mind-map в текстовом виде** — иерархия, связи
- [ ] **Отдельный файл** — DOCX, PDF, EPUB (если есть техническая возможность)
- [ ] **Структура / оглавление** — иерархический список
- [ ] **Список / перечень** — просто список
- [ ] **Аннотация / краткое описание** — 1-2 абзаца
- [ ] **Развёрнутый ответ с пояснениями** — подробно, с обоснованиями

**Требования к оформлению (важно!):**
- **Заголовки:** уровень 1 (#), уровень 2 (##), уровень 3 (###) — как используем?
- **Выделения:** жирный шрифт (** **), курсив (* *), капитель (не надо)
- **Списки:** маркированные (*), нумерованные (1.)
- **Таблицы:** нужны / не нужны / только если необходимо
- **Текстовые дубли таблиц:** нужны для аудио / не нужны

**Твой формат вывода (напиши после двоеточия):**
