### БЛОК 0. СЛУЖЕБНЫЕ ПАПКИ (создать перед началом работы)

При запуске нового книжного проекта **обязательно создайте следующую структуру папок**. Она обеспечит порядок и сохранит все материалы для будущих сессий.

---

#### 📁 Структура корневой папки проекта

```
📁 Название_проекта/
│
├── 📁 00. Управление/
├── 📁 01. Контекст и управление/
├── 📁 02. Шаблоны и запросы/
├── 📁 03. Архив сессий/
├── 📁 04. Главы/
├── 📁 05. Приложения/
└── 📁 06. Финальные файлы/
```

---

#### 📁 00. Управление (самые важные документы под рукой)

Эта папка — **пульт управления проектом**. Здесь лежит только самое необходимое для быстрого доступа.

| Файл | Назначение | Когда создаётся |
|------|------------|-----------------|
| `00_Мастер_индекс.md` | Полное описание проекта (цели, задачи, статусы) | В начале, обновляется регулярно |
| `01_Карта_проекта.md` | Структура, сроки, этапы | В начале |
| `02_Карта_папок_и_файлов.md` | Схема хранения | В начале |
| `03_Структура_книги.md` | Оглавление с главами | В начале |
| `04_График_работ.md` | План по дням/неделям | В начале |
| `05_Чек_лист_готовности.md` | Контроль качества | В начале + в конце |

---

|------|------------|

## КОНТРОЛЬНЫЙ ЧЕК-ЛИСТ ПРИ СТАРТЕ ПРОЕКТА

Перед началом работы над новой книгой:

- [ ] Создана корневая папка проекта
- [ ] Созданы все служебные папки (00–06)
- [ ] В папку `02. Шаблоны` скопированы все шаблонные файлы
- [ ] В папку `00. Управление` создан мастер-индекс
- [ ] Создана карта папок и файлов
- [ ] Утверждена структура книги
- [ ] Первая сессия готова к старту

---

## КОНТРОЛЬНЫЙ ЧЕК-ЛИСТ ПРИ ЗАВЕРШЕНИИ СЕССИИ

- [ ] Все новые главы сохранены в `04. Главы`
- [ ] Мастер-файл обновлён
- [ ] Напоминалка обновлена
- [ ] Утренний отчёт сохранён
- [ ] Все файлы скопированы в `03. Архив сессий / Сессия_N_дата`

---

## ГОТОВЫЙ ТЕКСТ ДЛЯ НОВОГО ПРОЕКТА

Чтобы не набирать каждый раз вручную, просто скопируйте этот текст в первый запрос новой книги:

```
Перед началом работы прошу создать следующую структуру папок:

00. Управление
01. Контекст и управление
02. Шаблоны и запросы
03. Архив сессий
04. Главы
05. Приложения
06. Финальные файлы

В папку 02 скопировать все шаблонные файлы из предыдущих проектов.
В папке 00 создать мастер-индекс и карту проекта.

Готов к работе.
```
