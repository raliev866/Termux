# ШАБЛОН ФОРМАТА ТЕКСТА
## Сокращённая книга: «ЖКТ — ключ к здоровью. 21 шаг к восстановлению»

---

## 1. ОБЩАЯ СТРУКТУРА ТЕКСТА

### 1.1. Иерархия заголовков

```
# Название книги (только в начале файла)
# Название части (Уровень 1)
## Название шага/дня (Уровень 2)
### Подзаголовок внутри шага (Уровень 3)
```

**Пример:**
```
# Часть 1. Диагностика за 3 дня

## День 1. Оцениваем кислотность желудка

### Простой тест с содой
### Тест с яблочным уксусом
### Дневник симптомов
```

---

## 2. ОФОРМЛЕНИЕ ТЕКСТА

### 2.1. Выделения

| Элемент | Оформление | Пример |
|---------|------------|--------|
| **Ключевые термины** | `**жирный**` | **Бетаин-HCl** принимают во время еды |
| *Акцент* | `*курсив*` | *Важно:* не превышайте дозировку |
| ***Особо важное*** | `***жирный курсив***` | ***Запомните: изжога ≠ высокая кислотность*** |

**Правила:**
- **Жирный шрифт** — только для ключевых терминов и важных понятий
- *Курсив* — для акцентов и предупреждений
- ***Жирный курсив*** — только для критически важных выводов (1 раз на шаг)

---

### 2.2. Списки

**Маркированный список (для советов, ингредиентов, пунктов):**
```
- Первый пункт
- Второй пункт
  - Подпункт (с отступом 2 пробела)
- Третий пункт
```

**Нумерованный список (для последовательных действий):**
```
1. Первый шаг
2. Второй шаг
   1. Подшаг 2.1
   2. Подшаг 2.2
3. Третий шаг
```

**Чек-лист (для самопроверки):**
```
- [ ] Не выполнено
- [x] Выполнено
```

---

### 2.3. Чек-листы в конце каждого шага

В конце каждого шага (главы) — **обязательный чек-лист**:

```
## ✅ Чек-лист: что сделать сегодня

- [ ] Выполнить тест с содой
- [ ] Записать результат в дневник
- [ ] Оценить своё самочувствие
```

---

### 2.4. Важные предупреждения

```
⚠️ **Важно:**
Никогда не превышайте рекомендованную дозировку. При появлении жжения или дискомфорта прекратите приём и проконсультируйтесь с врачом.
```

---

### 2.5. Советы и лайфхаки

```
💡 **Совет:**
Для лучшего эффекта принимайте магний вечером, за час до сна.
```

---

### 2.6. Таблицы (только самое нужное)

Таблицы используем только там, где без них действительно не обойтись. Каждая таблица должна сопровождаться кратким текстовым пояснением.

```
| Продукт | PRAL-индекс | Влияние |
|---------|--------------|---------|
| Огурцы | -2,5 | Ощелачивают |
| Мясо | +7,5 | Закисляют |

📋 **Кратко:** Овощи и зелень ощелачивают организм, мясо и зерновые — закисляют.
```

## 3. ТРЕБОВАНИЯ К ТЕКСТУ

### 3.1. Длина

- **Каждый шаг:** 1–2 страницы (не больше!)
- **Абзацы:** 2–4 предложения
- **Предложения:** короткие и средние (до 15 слов), никаких длинных конструкций

### 3.2. Язык

- Максимально простой и понятный
- Минимум терминов (каждый термин — с кратким пояснением)
- Никакой «воды» — только конкретика
- Обращение к читателю: **на «ты»**

### 3.3. Структура каждого шага

1. **Заголовок** (## Шаг 1. Название)
2. **Краткое введение** (1–2 предложения — зачем этот шаг)
3. **Основная часть** (инструкция, разбитая на подзаголовки ###)
4. **Чек-лист** (в конце)
5. **Предупреждение** (если нужно)

---

## 4. ПРИМЕР ИДЕАЛЬНОГО ШАГА

```
## Шаг 1. Как правильно принимать бетаин-HCl

Этот шаг — для тех, кто уже подтвердил пониженную кислотность и готов её компенсировать. Бетаин-HCl заменяет недостающую соляную кислоту и помогает переваривать белок.

### Кому можно, кому нельзя

❌ **Нельзя принимать при:**
- язве желудка и двенадцатиперстной кишки
- эрозивном гастрите в стадии обострения
- приёме обезболивающих (НПВС)

✅ **Можно, если:**
- кислотность подтверждена (тест с содой или гастроскопия)
- нет болей и жжения

### Схема подбора дозировки

1. **День 1–3:** по 1 капсуле во время самого плотного приёма пищи (где есть мясо)
2. **День 4–6:** по 2 капсулы (распределить на весь приём)
3. **День 7–14:** по 3 капсулы (по 1 на каждый приём белка)

### Контрольные признаки

- 🔥 Чувство жжения или тепла в желудке — **доза слишком велика**, уменьшить
- 😊 Легкость после еды, исчезновение тяжести — **доза подобрана правильно**
- 🤔 Эффекта нет — возможно, **доза недостаточна**, можно увеличить

### Сколько принимать

Обычно курс составляет 1–3 месяца. Затем пробуем снизить дозу или отменить — смотрим на самочувствие.

---

## ✅ Чек-лист: что сделать сегодня

- [ ] Убедиться, что нет противопоказаний
- [ ] Начать с 1 капсулы во время еды
- [ ] Записать свои ощущения
- [ ] Не превышать дозировку

⚠️ **Важно:** Если через 30–60 минут после приёма появилась изжога или жжение — значит, кислотность у вас в норме или повышена. Бетаин-HCl вам не нужен. Прекратите приём.
```

## 5. ЗАПРЕТЫ (ЧЕГО НЕ ДЕЛАТЬ)

- ❌ **Никакой теории** — только практика
- ❌ **Никаких длинных объяснений** — только инструкции
- ❌ **Никаких сложных терминов без объяснения**
- ❌ **Никаких ссылок на исследования** (это не научная монография)
- ❌ **Никаких имён** (Шишонин, Неумывакин и др.)
- ❌ **Никаких личных историй**
- ❌ **Никаких переходов к следующей главе** в конце
- ❌ **Никаких резюме** (только чек-лист)
