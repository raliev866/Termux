# Приложение 4. Диета Low FODMAP (подробные таблицы)

**Что такое FODMAP?**
Это аббревиатура, обозначающая группу ферментируемых углеводов: Fermentable Oligosaccharides, Disaccharides, Monosaccharides And Polyols (олигосахариды, дисахариды, моносахариды и полиолы). Эти вещества плохо всасываются в тонком кишечнике и быстро ферментируются бактериями, вызывая вздутие и газообразование у людей с СИБР и чувствительным кишечником.

**Структура диеты:**
1. **Фаза исключения** (2–6 недель) — убрать все продукты с высоким содержанием FODMAP
2. **Фаза введения** (6–8 недель) — по одному возвращать продукты, отслеживая реакцию
3. **Фаза поддержания** — индивидуальный рацион с исключением только тех продуктов, которые вызывают симптомы

## Таблица 4.1. Продукты с высоким и низким содержанием FODMAP

### Овощи

| Высокий FODMAP (исключить) | Низкий FODMAP (разрешено) |
|-----------------------------|---------------------------|
| Чеснок, лук (все виды) | Зелёный лук (только зелёная часть) |
| Цветная капуста | Брокколи (только соцветия, до 75 г) |
| Белокочанная капуста | Морковь |
| Брюссельская капуста | Огурцы |
| Грибы | Кабачки, цукини |
| Спаржа | Картофель |
| Артишоки | Помидоры (до 2 ст.л. томатной пасты) |
| Свёкла (сырая) | Свёкла (варёная, до 70 г) |
| Сельдерей | Болгарский перец |
| Горошек зелёный | Шпинат, салат, руккола |

📋 **Для прослушивания:**
В фазе исключения убираем все виды лука и чеснока, цветную и белокочанную капусту, грибы, спаржу. Оставляем морковь, огурцы, кабачки, картофель, помидоры, зелень и болгарский перец.

### Фрукты

| Высокий FODMAP (исключить) | Низкий FODMAP (разрешено) |
|-----------------------------|---------------------------|
| Яблоки (особенно красные) | Бананы (зелёные или плотные) |
| Груши | Апельсины, мандарины |
| Арбуз, дыня | Виноград |
| Персики, нектарины | Клубника, малина, голубика (до 30–40 г) |
| Сливы | Киви (1 шт) |
| Манго | Ананас |
| Авокадо (больше ⅛ шт) | Лимон, лайм |
| Сухофрукты (финики, курага, изюм) | Ревень |

📋 **Для прослушивания:**
Из фруктов разрешены цитрусовые, виноград, ягоды в небольшом количестве, зелёные бананы. Яблоки, груши, арбуз и косточковые временно исключаются.

### Молочные продукты

| Высокий FODMAP (исключить) | Низкий FODMAP (разрешено) |
|-----------------------------|---------------------------|
| Молоко коровье (лактоза) | Молоко безлактозное |
| Йогурты с лактозой | Твёрдые сыры (чеддер, пармезан, гауда) |
| Мягкие сыры (рикотта, творог) | Сливочное масло |
| Сметана | Кокосовое молоко (консервированное) |
| Мороженое | Рисовое, миндальное молоко |

📋 **Для прослушивания:**
Лактоза — высокий FODMAP. Поэтому обычное молоко, мягкие сыры и йогурты заменяем на безлактозные аналоги. Твёрдые выдержанные сыры безопасны.

## Пример меню на день (фаза исключения)

**Завтрак:**
- Овсяная каша на безлактозном молоке или воде
- Горсть черники
- Зелёный чай

**Обед:**
- Куриная грудка, запечённая с травами (без чеснока и лука!)
- Гарнир из отварного риса
- Салат из огурцов, помидоров и зелени с оливковым маслом

**Ужин:**
- Запечённая рыба с лимоном
- Отварной картофель с укропом
- Тушёные кабачки

**Перекусы:**
- Апельсин
- Горсть грецких орехов (до 30 г)
- Рисовые хлебцы
