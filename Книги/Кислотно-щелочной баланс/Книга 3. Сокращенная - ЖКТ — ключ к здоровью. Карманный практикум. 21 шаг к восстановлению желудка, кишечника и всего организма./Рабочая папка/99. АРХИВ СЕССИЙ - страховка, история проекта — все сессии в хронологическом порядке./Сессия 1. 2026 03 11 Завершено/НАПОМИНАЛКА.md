
# Напоминалка — то же что и мастер-файл контекста, но очень кратко, на один экран.  

# Шаблон напоминалка — рамка для создания напоминалки нового проекта.

# **Напоминалка** — это сверхкраткая версия мастер-файла. Она нужна для быстрого старта, когда не хочется читать много страниц. Использовать её можно в начале новой сессии, чтобы быстро напомнить мне главное. В конце сессии её тоже надо обновлять. Папка правильная.

# **Шаблон напоминалка** — это готовая структура для создания краткой напоминалки для любого проекта. Использовать его нужно при запуске нового проекта. Папка правильная.




# Шаблон-напоминалка — это сверхкраткий документ, который содержит только самую критическую информацию для быстрого старта новой сессии. Это не мастер-файл (там всё подробно), а именно шпаргалка, напоминание о главном.
